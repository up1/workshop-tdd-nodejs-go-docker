# Develop and Testing API with NodeJS
* Express
* Database with PostgreSQL + Sequelize
* Testing
  * Jest
  * SuperTest
  * Nock

## Create and install dependencies
```
$npm init
$npm i -S express jsonwebtoken sequelize pg
```

## Start Server
```
$npm start
```

## Testing with Jest
```
$npm i -D jest supertest nock

$npm test
```
